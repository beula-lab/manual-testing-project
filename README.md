# Manual Testing Project

## Project Overview
This project demonstrates manual testing of an e-commerce web application.

## Modules Covered
- Login
- Product Listing
- Add to Cart
- Checkout

## Testing Artifacts
- Test Scenarios
- Test Cases
- Bug Reports
- RTM (Requirement Traceability Matrix)

## Tools Used
- Microsoft Excel

## Key Highlights
- Designed test cases covering positive and negative scenarios
- Identified and documented bugs with severity and priority
- Maintained traceability between requirements and test cases
